■使い方

1)
参照があるか調べたいアセットファイルを選択して[Target AssetPath]ボタンを押してください。

2)
参照があるか調べたい範囲のフォルダまたはファイルを、１つまたは複数選択して、
[Increase All]ボタンを押してください。

※注意:
　デフォルトで参照対象に Assets フォルダが設定されていますが、
　これだとプロジェクト全体に対して参照の検査をかけてしまい、
　参照の検査で長時間待たされる事になりますので、
　予め参照があるであろう範囲がわかっている場合は、
　Assets の左の[-]ボタンを押して、
　参照対象から Assets フォルダを除外してください。


3)
Target AssetPath と Search TargetPaths が設定されると、
Target AssetPath に設定されたアセットファイルを参照している箇所を、
Search TargetPaths で指定した範囲から検査し、
参照があればリストに表示します。



-------------------------------------------------------------------------------
●検出された参照リストの操作について

[Detail]ボタン
　参照の詳細な情報を表示します。

[>]ボタン
　参照を保持しているアセットファイルを選択状態にします。


●参照の詳細情報について

[AssetType]
　参照を保持しているアセットファイルの大まかな種類です。
　以下の３種類があります

　　・Asset   マテリアルやテクスチャなどのプリミティブなアセットファイルです
　　・Prefab  プレハブファイルです
　　・Scene 　シーンファイルです

[HierarchyPath]
　アセットファイルがプレハブかシーンの場合に、
　参照を保持しているヒエラルキー上のパスを表示します。

　[>]ボタン
　　プレハプまたはシーンを展開し、参照を保持しているゲームオブジェクトを選択状態にします。

[ComponentType]
　参照を保持しているコンポーネントの種類です。

[PropertyType]
　参照を保持しているコンポーネント内のプロパティの種類です。

[PropertyPath]
　参照を保持しているコンポーネント内のプロパティの変数名(パス)です。

■補足:

プロジェクトタブ内のアセットファイルを選択した状態で、
右クリックしてポップアップメニューを表示し、

　SimpleReferenceDitector

を選択しても、本ツールを起動する事が出来ます。
